텍스트 스타일

        선택자{
            color:색상명영문이름|16진수숫자|rgb|rgba|hsl|hsla;
        }

색상값 참조사이트: 사이트로이동

  • 색상명으로 지정
  • 16진수값으로 지정
  • rgb로 지정
  • rgba로 지정
  • hsl로 지정
  • hsla로 지정

text-decoration 속성 : 텍스트에 줄을 긋거나 줄을 없앨때 사용

        선택자{
            text-decoration:none|underline|overline|line-through
        }
  • 하이퍼링크 밑줄 없애기
  • 텍스트영역 아래 줄 긋기
  • 텍스트영역 중간에 줄 긋기
  • 텍스트영역 위에 줄 긋기
